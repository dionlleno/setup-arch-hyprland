### Configurando o Hyprland

```
dotfiles/hyprland/
└── .config/hypr/
├── hyprland.conf # arquivo principal
├── env.conf # variáveis de ambiente
├── monitors.conf # monitores
├── input.conf # teclado, mouse, touchpad
├── general.conf # gaps, layout, bordas
├── decorations.conf # blur, shadow, rounding
├── animations.conf # animações
├── layouts.conf # dwindle / master
├── rules.conf # window rules
├── keybinds.conf # atalhos
├── autostart.conf # apps no boot
└── exec.conf # exec-once / scripts
```
